# Notes Management Web Application

This is a simple web application built using Flask that allows users to create and manage their notes.

## Features
- Add new notes
- View existing notes
- Edit notes
- Delete notes

## Technologies Used
- Python
- Flask
- HTML, CSS
- SQLite

## How to Run

You can run this project using VS Code or any Python IDE.

1. Install Flask:
   pip install flask

2. Open the project folder in VS Code

3. Run the application:
   python app.py

4. Open your browser and go to:`http://127.0.0.1:5000`

## About the Project
This project was created to learn the basics of web development using Flask, including routing, templates, and database integration.
