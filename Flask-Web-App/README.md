# Notes Management Web Application

This is a web application built using Flask that allows users to create, manage, and organize their notes efficiently. It demonstrates my understanding of basic web development concepts including backend development and database integration.

## Features
- Add new notes
- View existing notes
- Edit notes
- Delete notes
- Persistent storage using SQLite database

## Technologies Used
- Python
- Flask
- HTML, CSS
- SQLite

## How to Run

This project can be run using VS Code or any Python IDE.

1. Install required dependency:
   pip install flask

2. Open the project folder

3. Run the application:
   python main.py

4. Open your browser and go to:
   http://127.0.0.1:5000

## Description
I used Flask as the backend framework and SQLite as the database to store notes. The application supports full CRUD (Create, Read, Update, Delete) operations, allowing users to manage their notes dynamically.

## About the Project
This project helped me understand core web development concepts such as routing, templates, form handling, and database operations using Flask.
