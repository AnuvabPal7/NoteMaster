# Brain Tumor Detection using Machine Learning

This project focuses on detecting brain tumors using a machine learning approach. It was developed as a final year academic project to understand how machine learning techniques can be applied to real-world problems.

## Description
A Random Forest model is used to classify whether a tumor is present or not based on the dataset. The project includes data preprocessing, model training, and evaluation of results.

## Features
- Data preprocessing and cleaning
- Feature handling and preparation
- Model training using Random Forest
- Prediction and evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Result
- Achieved approximately 95% accuracy
